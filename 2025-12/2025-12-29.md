## Spring Boot 画像取得
- Controllerの作成不要
- **静的リソース配信**の仕組みを使う
  - URLにアクセスされたら指定ディレクトリのファイルをそのまま返す
 
### 実装例
```java
@Configuration
@RequiredArgsConstructor
public class WebConfig implements WebMvcConfigurer {

    private final MediaProperties mediaProperties;

    @Override
    public void addResourceHandlers(ResourceHandlerRegistry registry) {
        registry.addResourceHandler(mediaProperties.getPublicPrefix() + "/**")
                .addResourceLocations("file:" + mediaProperties.getRootDir() + "/");
    }
}
```

- @Configuration : 設定クラスの定義
- implements WebMvcConfigurer : Spring MVC の設定を 上書き/追加するためのインターフェース
- addResourceHandlers : 「静的ファイル配信ルール」を追加
  - addResourceHandler : どのURLを対象にするか
  - addResourceLocations：どこからファイルを探すか
- file:とは
  - addResourceLocations は「どの種類の場所から探すか」を示す
  - classpath:/static/ → jar内の静的リソース
  - file:/data/media/ → OSのファイルシステム
